游戏设计原理：
使用两个HTML5的canvas标签来制作。
一个标签绘制背景、海葵、果实，一个标签绘制鱼、漂浮物、UI界面、特效圈，然后把这两个标签叠加到一起。

玩法：
1、海葵不断产生两种果实，一种橙色的（10分值），一种蓝色的（20分值），并向上漂浮；
2、大鱼吃果实，当检测到大鱼和果实碰撞时即吃果实成功，并伴有白色的特效圈，同时大鱼的身体颜色发生变化；
3、大鱼喂小鱼，当检测到大鱼和小鱼碰撞时即喂果实成功，并伴有红色的特效圈，同时小鱼的身体颜色发生变化；
4、小鱼满血状态时身体颜色是红色，如果不喂食则它的身体颜色慢慢变浅，当身体颜色变为白色时，小鱼死亡，游戏结束；
5、所以大鱼需要不断地吃果实喂小鱼，以维持小鱼的血量，保持小鱼的身体颜色为红色。
